## 🇬🇧 ENGLISH

MOTOR doesn't mean "machine" — it means *the thing that stirs.*

**أثار** (athāra) = to stir, to agitate, to set in motion
**متور** (matwūr) = that which rotates / the driven thing

متور → matwūr → motor → **MOTOR**

Rule 006: Arabic ث shifts to T in Latin transmission.
The م prefix becomes the M- in Motor.

🧩 Strong evidence — Lisān al-ʿArab + phonosemantic analysis.

Same root family: agitate, motivate, stir — all words for *motion*.
The concept existed in Arabic before Latin built the word.

**أثار** = التحريك، الإثارة، الإدارة
**متور** = ما يدور / ما يُحرَّك

متور ← matwūr ← motor ← **MOTOR**

## 🇫🇷 FRENCH

MOTOR ne signifie pas "machine" — il signifie *ce qui agite.*

**أثار** (athāra) = agiter, mettre en mouvement
**متور** (matwūr) = ce qui tourne / ce qui est mû

متور → matwūr → motor → **MOTOR**

Règle 006 : le ث arabe devient T lors du passage en latin.
Le préfixe م devient le M- de Motor.

🧩 Preuves solides — Lisān al-ʿArab + analyse phonosémantique.

Même famille de racines : agiter, motiver — tous des mots pour *le mouvement*.
