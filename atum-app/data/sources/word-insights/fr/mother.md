# mère

**Racine arabe :** أُمّ (A-M-M) — origine, source, ce dont l'existence s'est rassemblée
**Racine cosmique :** ATUM — le centre, l'origine, la première masse
**Confiance :** 🔬

---

## Le Sens Profond

*Mère* ← proto-indo-européen *\*māter* ← la racine cosmique *\*MA*.
L'arabe *umm* = l'origine de tout : *umm al-qurā* (mère des cités), *umm al-kitāb* (mère du livre), *umm al-ra's* (mère de la tête = cerveau).
La racine *\*MA* est documentée dans 847+ langues sur tous les continents. 🔬

---

## La Physique

La mère = point d'origine, premier centre de gravité = ATUM.
*\*MA* = mère + eau + matière (matter) — trois concepts d'une seule racine.
La matière primordiale = le milieu qui porte l'énergie = ATUM dans tous ses sens.

## Dans Votre Vie Quotidienne

Mère / Madre / Mutter / Mãe / Māma / أُمّ.
Le son "ma" est le premier son que produisent les mammifères — tous.
Ce n'est pas une coïncidence — c'est la racine la plus ancienne de l'histoire linguistique humaine. 🔬

## Pourquoi Cela Vous Importe

*Umm* (origine) + *umma* (nation) + *amām* (devant) + *imām* (chef) — tous d'une seule racine.
ATUM = ce qui était en premier, ce dont tout est venu.
